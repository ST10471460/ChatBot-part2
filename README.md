GUI chatbot based on cyber security


What is this program:
This application is a graphical user interface that expands on a Cybersecurity Awareness Chatbot.
This chatbot should be more dynamic, recognise specific cybersecurity related keywords and incorporate memory to recall information shared.
It will have sentiment detection enabling the bot to adjust responses based on the users tone, enabling the bot to be more interactive 
and simulate a natural conversation.
